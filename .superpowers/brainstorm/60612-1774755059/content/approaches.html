3 Approaches: Layered Character System

All use runtime compositing (OffscreenCanvas) + DSL/config driven. They differ in how layers are structured and assets managed.

A

Full Spritesheet Layers

Each layer variant is a complete 7x3 spritesheet (same grid as today). Composed into one final spritesheet at load time.

Pros

  • Simplest compositing — just overlay PNGs pixel by pixel
  • Each layer is fully self-contained
  • Easy to hand-draw or AI-generate

Cons

  • More total pixels to draw (21 frames per variant)
  • Alignment must be pixel-perfect across all frames

Effort: ~2 weeks. ~10 base spritesheets to draw. Compositing is straightforward.

B

Region-Based Layers + Palette Swap

Layers are partial spritesheets (only the pixels they affect). Body has skin-tone regions mapped to a palette. Clothing layers use indexed colors swapped at compose time.

Pros

  • Fewer pixels to draw — hair layer only has hair pixels
  • Color variations are free (palette swap)
  • Smallest asset footprint

Cons

  • More complex compositing (transparency masking + palette remapping)
  • Asset pipeline needs strict palette discipline

Effort: ~3 weeks. Less drawing, more code. Palette system adds complexity.

C

Hybrid: Full Sheets + Simple Recolor

Layer variants are full 7x3 spritesheets (like A), but each uses a small indexed palette (3-4 colors). At compose time, palette colors are swapped for the chosen colors. Best of both worlds.

Pros

  • Simple compositing (overlay like A)
  • Color variations via palette swap (like B)
  • Asset creation is straightforward
  • Recolor logic is ~30 lines of code

Cons

  • Still need full spritesheets per shape variant
  • Palette must be consistent across frames

Effort: ~2 weeks. Same drawing as A, recolor adds ~1 day of code.

Recommendation: C (Hybrid)

Same simplicity as A for compositing and asset creation, but palette swap gives you infinite color variations for free. A hair_short.png base spritesheet can be blonde, brown, red, blue — just by swapping 3 colors. Minimal extra code, massive variety boost.
